# Stock Portfolio Tracker

## Project Description
This is a simple Stock Portfolio Tracker developed using Python. The program allows users to enter stock names and quantities, then calculates the total investment value using predefined stock prices.

## Features
- User-friendly console interface
- Predefined stock prices
- Calculates total portfolio value
- Accepts multiple stock entries
- Displays investment summary
- Handles invalid stock names

## Technologies Used
- Python 3
- VS Code

## Concepts Used
- Dictionary
- Variables
- User Input
- Data Type Conversion (`int()`)
- For Loop
- If-Else Statements
- Dictionary Lookup
- Arithmetic Operations
- String Methods (`upper()`)
- Formatted Output (f-strings)

## How to Run
1. Open the project in VS Code.
2. Run the `stock_portfolio.py` file.
3. Enter the number of different stocks you own.
4. Enter the stock name and quantity.
5. View the total portfolio value.
